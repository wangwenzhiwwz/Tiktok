# 📱 TikTok 风格短视频演示页面 (Ulefone 主题)

这是一个使用纯 HTML、CSS 和 JavaScript 构建的简易短视频垂直滚动演示页面。它模仿了 TikTok 或 Douyin 的核心交互模式，包括垂直滚动切换、自动播放/暂停、视频信息侧边栏和随机数据生成。

---

## ✨ 核心功能

* **垂直沉浸式滚动**：使用 CSS `scroll-snap` 实现，确保每次滚动都能精准定位到下一个视频。
* **不静音设置**：视频默认不静音（`video.muted = false`）。
    * **⚠️ 注意：** 根据现代浏览器自动播放策略，此设置会导致视频无法自动播放，需要用户手动点击。
* **内容自动加载**：项目通过脚本自动生成 `video/1.mp4` 到 `video/15.mp4` 的路径，模拟内容流。
* **随机 Ulefone 主题内容**：
    * 视频标题和描述是关于 **Ulefone（欧乐风）手机** 的随机主题内容。
    * 点赞、评论、收藏和分享等数据采用随机生成，并以 K（千）/ 万 / M（百万）格式显示。
* **播放/暂停控制**：点击视频主体区域可切换播放/暂停状态，并显示/隐藏一个大型播放图标。
* **进度条**：顶部的细进度条显示当前视频的播放进度。

---

## 🛠️ 如何安装与运行

本项目是一个静态 HTML 页面，无需任何服务器环境即可运行。

1.  **下载文件**：将您获得的 `index.html` 文件保存到本地。
2.  **创建视频文件夹**：在 `index.html` 文件所在的目录下，创建一个名为 `video` 的文件夹。
3.  **准备视频文件**：将您的 MP4 视频文件放入 `video` 文件夹中，并确保文件名是连续的数字格式：
    * `1.mp4`
    * `2.mp4`
    * `3.mp4`
    * ... 直到 `15.mp4`
4.  **运行**：双击 `index.html` 文件，它将在您的默认浏览器中打开。

## 💡 使用说明与重要注意事项

### 1. 自动播放限制

由于您要求**“不要静音”**，浏览器会阻止视频在滚动进入视口时自动播放。

* **解决方案**：请**手动点击**视频主体区域。第一次点击将播放视频并发出声音。
* 如果您希望恢复自动播放，请将 `index.html` 文件中所有 `video.muted = false` 改回 `video.muted = true`。

### 2. 交互方式

* **上下滚动**：切换不同的视频内容。
* **点击视频**：播放/暂停当前视频。

### 3. 内容修改

如果您想修改随机生成的 Ulefone 标题列表，请编辑 `index.html` 文件中 `<script>` 标签内的 `randomTitles` 数组。
